# MahoDex

Unofficial MangaDex reader client for J2ME platform.

Compatible with MangaDex API 5.10.2

![](res/d.svg)

[Proxy](https://github.com/shinovon/nnp-proxies)

## Features

- Manga reader
- Updates feed
- Advanced search
- Authorization for personal library and feed

## Device support

### Requirements

- MIDP 2.0
- CLDC 1.1
- JSR-75
