# MahoDex

Unofficial MangaDex reader client for J2ME platform.

Compatible with MangaDex API 5.13.1

![](res/d.svg)

[Proxy source](https://github.com/shinovon/nnp-proxies)

## Features

- Manga reader
- Updates feed
- Advanced search
- Authorization for personal library, read status and feed

## Device support

### Requirements

- MIDP 2.0, CLDC 1.1, JSR-75
- Heap memory: at least 2 MB
- Screen size: at least 176x220
- M3G (Optional, for precise resizing on client side)

## State

The project is considered complete and will not receive feature updates.
